# Descargador de YouTube

Aplicacion grafica para descargar videos de youtube usando yt-dlp de forma mas comoda.

## Requisitos
- Python 3.x
- Ejecutable de yt-dlp en el directorio de descarga

## Uso
1. Ejecuta `python youtube_downloader.py`
2. Ingresa un enlace de YouTube
3. Selecciona solo audio (opcional)
4. Especifica un nombre de archivo (opcional)
5. Elige un directorio de descarga que contenga yt-dlp
6. Haz clic en "Descargar"

## Características
- Descarga videos o audio MP3
- Nombre de archivo personalizado
- Selección de directorio con rutas en formato de barra invertida
- Manejo de errores para entradas inválidas

### sobre errores
es una aplicacion asi nomas para descargar mas facil los videos. si alguien mas la usa y encuentra algun error grave puede ser que nunca lo arregle
